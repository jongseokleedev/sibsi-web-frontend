# SibSi-web-backend: Fundraising assistant for Bday

### Description
- Sib-Si-Il-Ban is a service that helps you fundraise for events.

### Environment

- go version 1.19.5
- go gin web server
- MongoDB

### How To Start

1. env setting
   - Please see `.env.example` file and create `.env` file with the appropriate values.


2. execute main.go

    - `go run main.go`
